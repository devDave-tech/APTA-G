# APTA-G Scientific

Futuristic marketing site for **APTA-G Scientific** — custom DNA and peptide aptamers for research, diagnostics, and therapeutics.

## Stack

- **Vite** + **React 19** + **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (scroll reveals, mobile nav, FAQ)
- Canvas 2D molecular helix (no WebGL dependency)

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
